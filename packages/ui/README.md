# Noteece UI

This package contains shared UI components for the Noteece application.
It provides a consistent look and feel across the Desktop and potentially other web-based interfaces.

## Structure

- `src/`: Source code for React components.
- `package.json`: Dependencies and scripts.

## Usage

This package is consumed by `apps/desktop` and potentially other apps in the monorepo.

```bash
import { Button } from '@noteece/ui';
```

## Development

Run `npm run lint` to lint the code.
