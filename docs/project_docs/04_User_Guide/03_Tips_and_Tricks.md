# Tips and Tricks

Power user techniques to maximize your Noteece experience.

---

## Note-Taking Tips

### 1. Use Consistent Naming

Adopt a naming convention for notes:

- `YYYY-MM-DD - Daily Note`
- `Project - Topic`
- `Area - Subtopic`

### 2. Leverage Tags Strategically

Create a tag taxonomy:

```
#status/active
#status/archived
#type/meeting
#type/idea
#priority/high
```

### 3. Build a Knowledge Graph

- Create **hub notes** that link to related topics
- Use `[[WikiLinks]]` liberally
- Review backlinks to discover connections

### 4. Quick Capture Everything

- Capture first, organize later
- Use the Quick Capture shortcut (`Cmd/Ctrl + Shift + N`)
- Process your inbox during weekly review

## Task Management Tips

### 1. Use the 2-Minute Rule

If a task takes less than 2 minutes, do it immediately.

### 2. Prioritize with Eisenhower Matrix

| Urgent & Important         | Important, Not Urgent |
| -------------------------- | --------------------- |
| Do First                   | Schedule              |
| **Not Urgent & Important** | **Neither**           |
| Delegate                   | Eliminate             |

### 3. Break Down Large Tasks

- Create subtasks for anything taking >30 minutes
- Link tasks to projects
- Set realistic due dates

---

## Project Management Tips

### 1. Use Milestones

Break projects into phases:

1. Planning
2. Execution
3. Review
4. Completion

### 2. Track Risks

Use the **Risks** tab in Project Hub:

- Identify potential blockers
- Assign mitigation strategies
- Review weekly

### 3. Review with Timeline

Use the **Timeline** view for:

- Visualizing project progress
- Identifying bottlenecks
- Planning resource allocation

## Keyboard Mastery

### Navigation Speed

| Action           | Shortcut                  |
| ---------------- | ------------------------- |
| Jump to any note | `Cmd/Ctrl + K`, type name |
| Recent notes     | `Cmd/Ctrl + E`            |
| Switch modules   | `Cmd/Ctrl + 1-9`          |

### Editor Speed

| Action          | Shortcut               |
| --------------- | ---------------------- |
| Create heading  | `Cmd/Ctrl + 1-6`       |
| Create list     | `Cmd/Ctrl + Shift + L` |
| Toggle checkbox | `Cmd/Ctrl + Enter`     |

## AI Assistant Tips

### Effective Prompts

**Summarization:**

> "Summarize my notes from last week's project meetings"

**Task Generation:**

> "Create a task list for implementing the new feature based on my notes"

**Analysis:**

> "What patterns do you see in my productivity notes?"

### Context Matters

- Reference specific notes or time periods
- Be specific about output format
- Iterate on responses

## Sync Optimization

### Keep Devices in Sync

- Enable background sync on mobile
- Sync before switching devices
- Check sync status regularly

### Conflict Prevention

- Avoid editing the same note on multiple devices simultaneously
- Complete edits before switching devices
- Use sync status indicator

---

## Performance Tips

### Keep the App Fast

- Archive old projects
- Use tags instead of many folders
- Enable virtualization for large lists

### Reduce Clutter

- Archive completed projects monthly
- Delete unused notes quarterly
- Maintain your tag taxonomy

---

## Security Best Practices

### Protect Your Vault

1. Use a strong, unique password
2. Enable biometric unlock on mobile
3. Keep your devices updated
4. Backup your vault regularly

### Backup Strategy

- Export data weekly to encrypted backup
- Store backups in multiple locations
- Test restore process quarterly
