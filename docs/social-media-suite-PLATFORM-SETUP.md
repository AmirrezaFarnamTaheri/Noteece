## Twitter / X

### Account Setup

**Step 1: Add Account in Noteece**
1. Open Social Hub
2. Click "Add Account"
3. Select "Twitter" from platform dropdown
4. Enter your Twitter handle (without @)
5. Click "Connect"

**Step 2: Authenticate**
1. A WebView window opens to twitter.com
2. Log in with your credentials
3. If prompted, complete 2FA verification
4. Wait for timeline to load

**Step 3: Verify Syncing**
- Check that posts appear in Noteece timeline
- Look for sync status: üü¢ Active
- Posts sync every 60 minutes by default

### What Gets Synced

‚úÖ **Included:**
- Timeline tweets (your follows)
- Retweets and quote tweets
- Replies in timeline
- Likes, shares, views counts
- Images and videos
- Author information

‚ùå **Not Included:**
- Direct messages (private)
- Drafts
- Bookmarks
- Lists (separate view)

### Best Practices

- **Sync Frequency**: 30-60 minutes recommended
- **Privacy**: Don't sync if account has sensitive DMs
- **Multiple Accounts**: Name them clearly (e.g., "Personal Twitter", "Work Twitter")
- **Focus Mode**: Use "Deep Work" to block during work hours

### Troubleshooting

**Problem: No tweets appearing**
- Solution: Make sure timeline is visible, scroll to load posts
- Check that account isn't private/protected

**Problem: Sync fails with üî¥ error**
- Solution: Re-authenticate by clicking account ‚Üí "Reconnect"

**Problem: Old tweets missing**
- Solution: Twitter API limits - only recent tweets synced

### What Gets Synced

‚úÖ **Included:**
- Match names (FIRST NAMES ONLY)
- Conversation previews
- Match timestamps
- Profile photos (limited)

‚ùå **Not Included:**
- Full names (privacy)
- Phone numbers
- Addresses

### Privacy Protections

**Built-in Safeguards:**
- üîí Only first names stored
- üîí Tagged as "sensitive" metadata
- üîí Less frequent syncing (60 min default)
- üîí Explicit user consent required
- üîí Privacy notice displayed on first use

### Best Practices

- **Sync Frequency**: 60-120 minutes (or disable entirely)
- **Device Security**: Enable full disk encryption
- **Sharing**: Never share device with dating apps synced
- **Focus Modes**: Use "Learning" mode to block during work
- **Consider Disabling**: Seriously consider if you need this synced

### Troubleshooting

**Problem: Don't want dating apps synced anymore**

- Or disable account to pause syncing

## Troubleshooting

### General Issues

**Problem: WebView won't load**
```
Solutions:
1. Check internet connection
2. Clear cookies: Settings ‚Üí Clear Session Data
3. Restart Noteece
4. Check firewall isn't blocking
```

**Problem: Sync status shows üî¥ Error**
```
Solutions:
1. Click account ‚Üí "Reconnect"
2. Re-authenticate in WebView
3. Check platform isn't down
4. Verify credentials are correct
```

**Problem: Posts not appearing in timeline**
```
Solutions:
1. Check platform filter (might be hidden)
2. Verify time range filter
3. Wait for sync cycle to complete
4. Check that account is enabled
```

**Problem: Slow performance**
```
Solutions:
1. Close unused WebView windows
2. Reduce number of active accounts
3. Lower sync frequency
4. Enable old post cleanup
```
