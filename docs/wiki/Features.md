# Features Guide

## Knowledge Management

### Notes
- **Markdown Support:** Full GFM support including tables, task lists, and code blocks.
- **Backlinks:** Link notes using `[[WikiLink]]` syntax. The "Backlinks" panel shows referencing notes.
- **Graph View:** Visual representation of the knowledge graph (nodes = notes, edges = links).
- **Versioning:** Every save creates a snapshot. View and restore history from the "History" tab.

### Search
- **Full-Text Search:** Fast search across all note content.
- **Filters:**
    - `tag:work` - Filter by tag.
    - `created:2024-01-01` - Filter by date.
    - `type:note` - Filter by entity type.
- **Saved Searches:** Save complex queries as "Smart Folders".

## Project Hub

### Projects
- **Metadata:** Status (Active, On Hold, Done), Start Date, Target End Date.
- **Milestones:** Key checkpoints with dates.
- **Risks:** Track potential issues with Likelihood/Impact matrix.

### Tasks
- **Kanban Board:** Drag-and-drop tasks between statuses (Todo, In Progress, Done).
- **List View:** Sortable list for bulk management.
- **Recurrence:** Powerful recurring rules (e.g., "Every 2nd Tuesday").
- **Subtasks:** Break tasks down into smaller steps.

## Personal Growth

### Habit Tracker
- **Daily/Weekly Habits:** Set frequency.
- **Streaks:** Visual fire indicators for consistency.
- **Stats:** Completion rates and calendar heatmap.

### Goals
- **SMART Goals:** Structured goal setting.
- **Linking:** Connect Projects and Habits to Goals to track progress.

### Weekly Review
- **Wizard:** A step-by-step interface to:
    1.  Review completed tasks.
    2.  Check habit consistency.
    3.  Reflect on journaling.
    4.  Plan the upcoming week.

## Universal Dashboard

- **Widgets:**
    - **Time Tracking:** Start/stop timers for tasks.
    - **Calendar:** View tasks and events.
    - **Health:** Input metrics (Weight, Sleep, Mood).
    - **Music:** (Mobile) Local music player integration.
- **Zen Mode:** Collapses sidebars for focus.

## Social Media Suite

- **Aggregator:** View feeds from multiple platforms in one place.
- **Private Scraping:** Uses local scraping logic (no API keys required for read-only).
- **Analytics:** Track your own posting habits and engagement (if logged in).
