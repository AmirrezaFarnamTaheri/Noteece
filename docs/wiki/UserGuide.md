# User Guide

## Getting Started

1.  **Download:** Get the latest release for your OS from the Releases page.
2.  **Install:** Run the installer (setup.exe, .deb, .dmg).
3.  **Create Vault:**
    - Launch Noteece.
    - Click "Create New Vault".
    - Choose a location on your disk.
    - **Important:** Set a strong password. If you lose this, your data is unrecoverable.

## Navigation

- **Sidebar:** Access primary modules (Notes, Projects, Habits, etc.).
- **Command Bar:** Press `Ctrl+K` (or `Cmd+K`) to open the quick switcher. Search for notes, tasks, or commands.
- **Settings:** Access preferences, sync, and backup via the gear icon.

## Syncing Devices

1.  **Desktop:** Go to Settings > Sync.
2.  **Mobile:** Open Noteece Mobile > Settings > Sync.
3.  **Pair:**
    - Desktop: Click "Pair New Device". A QR code/PIN will appear.
    - Mobile: Scan QR code or enter PIN.
4.  **Sync:** Once paired, devices will sync automatically when on the same WiFi.

## Tips & Tricks

- **Daily Notes:** Use `Ctrl+D` to instantly open today's note. Great for journaling or scratchpad.
- **Linking:** Type `[[` to see a list of notes to link to.
- **Templates:** Create templates for recurring meeting notes or project structures in Settings > Templates.
- **Graph View:** Use the graph to find orphaned notes or unexpected connections.
