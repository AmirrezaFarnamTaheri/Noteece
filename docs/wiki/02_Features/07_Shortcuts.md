# Keyboard Shortcuts

Noteece is designed for keyboard-first efficiency.

## Global

| Windows/Linux      | macOS             | Action                           |
| :----------------- | :---------------- | :------------------------------- |
| `Ctrl + K`         | `Cmd + K`         | **Global Search** (Open Omnibar) |
| `Ctrl + ,`         | `Cmd + ,`         | Open Settings                    |
| `Ctrl + \`         | `Cmd + \`         | Toggle Sidebar                   |
| `Ctrl + Shift + L` | `Cmd + Shift + L` | Toggle Light/Dark Mode           |

## Navigation

| Windows/Linux | macOS     | Action            |
| :------------ | :-------- | :---------------- |
| `Alt + 1`     | `Cmd + 1` | Go to Dashboard   |
| `Alt + 2`     | `Cmd + 2` | Go to Notes       |
| `Alt + 3`     | `Cmd + 3` | Go to Tasks       |
| `Alt + 4`     | `Cmd + 4` | Go to Project Hub |
| `Alt + Left`  | `Cmd + [` | Go Back           |
| `Alt + Right` | `Cmd + ]` | Go Forward        |

## Editor

| Windows/Linux      | macOS             | Action                        |
| :----------------- | :---------------- | :---------------------------- |
| `Ctrl + N`         | `Cmd + N`         | New Note                      |
| `Ctrl + S`         | `Cmd + S`         | Save (Auto-save is also on)   |
| `[[`               | `[[`              | Trigger WikiLink Autocomplete |
| `#`                | `#`               | Trigger Tag Autocomplete      |
| `/`                | `/`               | Trigger Slash Command Menu    |
| `Ctrl + B`         | `Cmd + B`         | Bold                          |
| `Ctrl + I`         | `Cmd + I`         | Italic                        |
| `Ctrl + Shift + C` | `Cmd + Shift + C` | Code Block                    |
| `Ctrl + Alt + T`   | `Cmd + Opt + T`   | Toggle Typewriter Mode        |

## Task Management

| Windows/Linux | macOS | Action                        |
| :------------ | :---- | :---------------------------- |
| `N`           | `N`   | New Task (in Task View)       |
| `Esc`         | `Esc` | Close Modal / Clear Selection |
