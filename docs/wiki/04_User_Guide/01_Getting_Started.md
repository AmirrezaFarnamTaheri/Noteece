## 1. First Launch

When you open Noteece for the first time, you'll be guided through vault creation:

### Create Your Vault

1. **Choose a Location:** Select where to store your encrypted vault file
2. **Set a Password:** Choose a strong, memorable password
   - âš ï¸ **Critical:** This password encrypts your master key
   - âš ï¸ There is **no password recovery** - if you lose it, your data is lost forever
   - We recommend using a password manager
3. **Confirm:** Re-enter your password to confirm

### Password Best Practices

- Use at least 16 characters
- Mix uppercase, lowercase, numbers, and symbols
- Consider a passphrase (e.g., "correct-horse-battery-staple")
- Store a backup in a secure location (password manager, safe deposit box)

## 3. Your First Steps

### Step 1: Create a Space

Spaces help organize your content into contexts (Work, Personal, Projects).

1. Go to **Settings â†’ Spaces**
2. Click **"New Space"**
3. Enter a name (e.g., "Personal")
4. Choose a color and icon
5. Click **Create**

### Step 2: Write Your First Note

1. Click the **+ New Note** button (or press `Cmd/Ctrl + N`)
2. Enter a title
3. Start writing in Markdown
4. Try these features:
   - Add a tag: `#getting-started`
   - Create a link: `[[Another Note]]`
   - Add a checkbox: `- [ ] My first task`

### Step 3: Set a Goal

1. Navigate to **Goals** in the sidebar
2. Click **"New Goal"**
3. Enter:
   - Title: "Learn Noteece"
   - Target: 7 (days of usage)
   - Category: Learning
4. Track your progress daily!

### Step 4: Create a Project

1. Go to **Project Hub**
2. Click **"New Project"**
3. Fill in the details:
   - Name: "Getting Started Guide"
   - Description: "Complete the Noteece onboarding"
   - Link to your Goal
4. Add tasks to your project

### Step 5: Explore the Dashboard

1. Click **Dashboard** in the sidebar
2. View your widgets (stats, tasks, calendar, etc.)
3. Customize by:
   - Opening **Settings â†’ Control Panel**
   - Enabling/disabling widgets
   - Choosing a preset (Minimal, Productivity, Health, Power User)

## 4. Syncing to Mobile

Keep your data synchronized across all your devices.

### Prerequisites

- Download Noteece Mobile from App Store or Google Play
- Both devices must be on the **same WiFi network**

### Pairing Steps

1. **Desktop:** Go to **Settings â†’ Sync â†’ "Pair New Device"**
2. A QR code will appear on screen
3. **Mobile:** Open Settings â†’ Sync â†’ "Scan QR Code"
4. Scan the QR code with your phone
5. Confirm the PIN code on both devices
6. Sync begins automatically!

## 5. Keyboard Shortcuts

Master these shortcuts to boost your productivity:

### Essential Shortcuts

| Shortcut       | Action            |
| -------------- | ----------------- |
| `Cmd/Ctrl + K` | Global Search     |
| `Cmd/Ctrl + N` | New Note          |
| `Cmd/Ctrl + T` | New Task          |
| `Cmd/Ctrl + ,` | Settings          |

| `Cmd/Ctrl + J` | Open AI Assistant |

| Shortcut               | Action                |
| ---------------------- | --------------------- |

| Shortcut         | Action            |
| ---------------- | ----------------- |
| `Cmd/Ctrl + 1-9` | Switch to module  |

| `Esc`            | Close modal/panel |

1. Go to **Settings â†’ Language**
2. Choose from 7 supported languages:

   - Japanese, Chinese, Persian (Farsi)

### Dashboard Customization

1. Go to **Settings â†’ Control Panel**
2. Enable/disable widgets
3. Choose a preset or customize individually
4. Drag widgets to reorder
