# Dashboard Guide

The Dashboard is your command center in Noteece, providing at-a-glance visibility into your workspace.

## Overview

The Dashboard features a customizable widget-based layout that adapts to your workflow.

## Widgets

| Widget | Description |

| **Quick Stats** | Overview of notes, tasks, and projects |
| **Due Today** | Tasks due today |

| **Calendar** | Mini calendar view |
| **Focus Timer** | Pomodoro timer for focused work |

## Customization

### Control Panel

Access the Control Panel via Settings â†’ Control Panel to:

1. **Enable/Disable Widgets** - Toggle individual widgets
2. **Use Presets** - Apply preset configurations:
   - **Minimal**: Clean, focused layout
   - **Productivity**: Task-focused widgets
   - **Health Focus**: Wellness tracking
   - **Power User**: All widgets enabled
3. **Search Widgets** - Find specific widgets
4. **Reset to Defaults** - Restore default configuration

### Widget Sizes

Widgets come in three sizes:
- **Small**: Compact, essential info only
- **Medium**: Standard size, detailed view
- **Large**: Expanded, comprehensive data

### Reordering

Drag and drop widgets to reorder them on the dashboard.

## Focus Mode

Enable Focus Mode to hide distractions and concentrate on a single task:

1. Click the Focus Mode toggle
2. Select your current task
3. Timer starts automatically
4. Non-essential widgets are hidden

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + D` | Go to Dashboard |
| `Ctrl/Cmd + Shift + F` | Toggle Focus Mode |
| `Ctrl/Cmd + J` | Open AI Chat |
| `Ctrl/Cmd + K` | Command Palette |

## Tips

1. **Start minimal** - Enable only essential widgets
2. **Use Focus Mode** - Reduce distractions when working
3. **Check AI Insights** - Get proactive recommendations
4. **Review weekly** - Use Weekly Progress to track trends
